Divide Two Integers

Original page

Note: Assume we are dealing with an environment that could only store integers within the 32-bit signed integer range: [−2^31, 2^31 − 1]. For this problem, if the quotient is strictly greater than 2^31 - 1, then return 2^31 - 1, and if the quotient is strictly less than -2^31, then return -2^31.

My solution

  1. Change both numbers to negative(can deal with -2^31). Log down whether they have different signs.
  2. Use num and sum to track the sum of num divisors. Increase sum by sum += sum until sum<dividend
  3. Let result+=last_num to save previous result. Reset dividend to remainder and repeat step 2 until no remainder or num-last_num == 1

Only one special case (dividend=-2147483648, divisor=-1) leads to overflow of number. Need to handle it separately.

class Solution:
    def divide(self, dividend: int, divisor: int) -> int:
        # Note: Assume we are dealing with an environment that could only store integers within the 32-bit signed integer range
        # special case
        if dividend == -2147483648 and divisor == -1:
            return 2147483647
        
        if (dividend < 0 and divisor > 0) or (dividend > 0 and divisor < 0):
            minus = True
        else:
            minus = False
        if dividend > 0:
            dividend = -dividend
        if divisor > 0:
            divisor = -divisor

        result = 0
        last_num = 0
        num = 1
        last_sum = 0
        cur_sum = divisor

        while True:
            while dividend - cur_sum <= 0:
                last_sum = cur_sum
                if -2147483648 - cur_sum > cur_sum:
                    last_num = num
                    break
                cur_sum += cur_sum
                last_num = num
                num += num
            result += last_num
            if num - last_num == 1 or cur_sum == dividend:
                break

            dividend -= last_sum
            last_sum = 0
            cur_sum = divisor
            last_num = 0
            num = 1

        result = result if not minus else -result

        return result

Analysis

Time: O(log_2(dividend/divisor) )

divisor*2^n = dividend

n = log_2(dividend/divisor)

Space: O(1)
