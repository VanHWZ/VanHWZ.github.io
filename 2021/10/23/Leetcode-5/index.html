Zigzag Conversion

Original page

My solution

Initialize a list containing numRows empty string: strings=[""] * numRows.

Traverse the input string, compute character’s y coordinate in zigzag arrangement. Add that character to the end of string[y].

Python code

class Solution:
    def convert(self, s: str, numRows: int) -> str:
        if numRows == 1:
            return s
        y = 0
        strings = ["" for _ in range(numRows)]
        for idx, c in enumerate(s):
            strings[y] += c
            if idx % (2 * numRows - 2) < numRows - 1:
                y += 1
            else:
                y -= 1

        return "".join(strings)

Analysis

Time: O(n)

  • Traverse the string takes O(n) time

Space: O(n)​

  • strings takes $O(n)$ space
