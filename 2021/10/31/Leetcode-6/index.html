Reverse Integer

Original page

My solution

  1. Return 0 when the input $x=0$ or $x=-2^{31}$
  2. Log the sign of x. Convert x to positive int
  3. Use python list as queue to enqueue the units digit of x. And update x with x // 10. Also log the number of digits of x in the process.
  4. If number of digits = 10, then check if the result goes outside the domain. Traverse the list and compare each digit to $2^{31}-1$ from lef to right. If result goes outside, then return 0.
  5. Dequeue the digits, add it to the units of the result and update result with result * 10

Python code

Analysis

Time: O(n)

  • Need to traverse the list for 2 to 3 times.

Space: O(n)​

  • Space of the queue.
