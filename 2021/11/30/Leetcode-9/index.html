Leetcode-Integer to Roman

2021-11-30

Integer to Roman

Original page

My solution

For each base, find the numeral and add correspondent roman numerals to the result.

Python code

class Solution:
    def intToRoman(self, num: int) -> str:
        signs = {1:"I", 5:"V", 10:"X", 50: "L", 100: "C", 500: "D", 1000: "M"}
        bases = [1000, 100, 10, 1]
        result = ""
        for idx, base in enumerate(bases):
            x = num // base
            num -= x * base
            if x == 9:
                result += "%s%s" % (signs[base], signs[bases[idx - 1]])
            elif x >= 5:
                result += signs[base * 5]
                for _ in range(x - 5):
                    result += signs[base]
            elif x == 4:
                result += "%s%s" % (signs[base], signs[base * 5])
            elif x > 0:
                for _ in range(x):
                    result += signs[base]
            else:
                continue
        return result

Analysis

Time: O(n), n is the number of bases.

Space: O(n) to store the result.​
