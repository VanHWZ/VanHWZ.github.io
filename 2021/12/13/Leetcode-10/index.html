Leetcode-3 Sum

2021-12-13
  • Leetcode

3 Sum

Original page

My solution

  1. Fix nums[i]，use Two Sum solution for the rest part.

    class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        length = len(nums)
        if length < 3:
            return []
        
        out = []
        
        for i in range(length-2):
            remain = -nums[i]
            temp = {}
            for j in range(i+1, length):
                temp[remain-nums[j]] = j
            for j in range(i+1, length):
                if nums[j] in temp and temp[nums[j]] != j:
                    r = sorted([nums[i], nums[j], nums[temp[nums[j]]]])
                    if r not in out:
                        out.append(r)
        return out

  2. Sort nums first. Use two pointers to find the two sum for each nums[i].

    class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        length = len(nums)
        if length < 3:
            return []
        
        nums = sorted(nums)
        out = set()
        
        for i, n in enumerate(nums):
            j = i+1
            k = length-1
            while n<=0 and j<k:
                s = n+nums[j]+nums[k]
                if s < 0:
                    j += 1
                elif s > 0:
                    k -= 1
                else:
                    out.add((n, nums[j], nums[k]))
                    j += 1
        result = []
        for t in out:
            result.append(list(t))
        return result

  3. Use same idea as 2nd solution. But try to remove repetition check.

    class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        length = len(nums)
        if length < 3:
            return []
        
        nums = sorted(nums)
        out = []
        
        for i, n in enumerate(nums):
            if i>0 and n == nums[i-1]:
                continue
            j = i+1
            k = length-1
            while n<=0 and j<k:
                s = n+nums[j]+nums[k]
                if s < 0:
                    j += 1
                elif s > 0:
                    k -= 1
                else:
                    out.append([n, nums[j], nums[k]])
                    j += 1
                    k -= 1
                    while j<k and nums[j] == nums[j-1]:
                        j += 1
        return out

Analysis

Time: O(n^2).

Space: O(n) to store the result.​
