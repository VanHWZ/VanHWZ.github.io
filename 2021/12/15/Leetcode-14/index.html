Leetcode-Remove Nth Node From End of List

Remove Nth Node From End of List

Original page

My solution

  1. Make a node list. Delete the n-th node directly.
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        nodes = []
        p = head
        while p:
            nodes.append(p)
            p = p.next
        if nodes[-n] == nodes[0]:
            head = head.next
        elif n == 1:
            nodes[-2].next = None
        else:
            prev = nodes[-n-1]
            prev.next = nodes[-n+1]
        return head
  1. Use two pointers. One pointer is n positions ahead of the other. Once the front pointer reach the end, delete the node of the former pointer.
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        p1 = p2 = head
        while n > 0:
            p2 = p2.next
            n -= 1
        while p2 and p2.next:
            p1 = p1.next
            p2 = p2.next
        if not p2:
            head = head.next
        else:
            p1.next = p1.next.next
        return head

Analysis

Time: Both are O(n).

Space:

  • O(n) to store all nodes
  • O(1)
